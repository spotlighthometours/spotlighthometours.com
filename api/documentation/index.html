API Documentation

Get tour information, photos, ID by MLS ID, create tours, get tour types, products, create orders, and much more!

How it works

Send a request and get a response in XML/JSON format

In order to use this API you will need an API key. You must append your API key to every query.
Copy

Some methods require additional parameters. Please reffer to each method below.

Tours

getphotos

http://www.spotlighthometours.com/api/tours/?method=getphotos&format=json&(id=theTourID/vid=theVendorID)

getusertours

http://www.spotlighthometours.com/api/tours/?method=getusertours&format=json&(userID=theUsersID/email=theUsersEmail)

gettouridbymlsid

http://www.spotlighthometours.com/api/tours/?method=gettouridbymlsid&format=json&id=theMLSID&state=theState

create

http://www.spotlighthometours.com/api/tours/?method=create&format=json
Parameters

POST required parameters to create a new tour.

  • desired_date (Desired shoot date)
  • desired_time (Desired shoot time)
  • additionalInstructions
  • mls
  • mls_provider
  • userEmail (This is used to find the userID)
  • propertyTypeID (Commercial=1, Farm=2, Land=3, Multi=4, Residential=5)
